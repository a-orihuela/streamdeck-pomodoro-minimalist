# Icons

Añade aquí los archivos de icono del pack.

## Especificaciones obligatorias (docs/icons.md)

| Propiedad | Valor |
|---|---|
| Tamaño | **144 × 144 px** exactos |
| Formatos estáticos | SVG, PNG, JPEG |
| Formatos animados | GIF, WEBP |
| Tamaño máximo | 1 MB por icono |
| Nombre de archivo | máx. 80 caracteres |

## Iconos animados

| Propiedad | Recomendación |
|---|---|
| Frame rate | 10–20 fps |
| Duración | máx. 5 segundos |
| Formato | GIF o WEBP |

## Distribución

Usa **Icon Pack Man** (https://iconpackman.elgato.com/) para generar el `.streamDeckIconPack` final.
